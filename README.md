# 개인블로그 Ok's Archive
![image](https://github.com/dhrgodms/playground/assets/53715875/db1d0a90-d0eb-4ca2-97f5-3fd9186cd3c8)

- CRUD가 가능하도록 만든 블로그
- 공부 및 개인 아카이브용으로 제작하여 로그인/회원가입 기능 無

<br>

## 개발 환경
- Backend : Java Spring boot, Gradle
- Frontend : React, Styled-components, daisy ui
- IDE : Intelli J

<br>

## How to Run
- frontend
  ```
  git clone
  cd src/main/my-app
  npm run start
  ```

- backend
  ```
  cd
  ./gralew bootRun
  ```
